# Console-Based Banking System

## Overview
A Java console application demonstrating Object-Oriented Programming (OOP) principles, designed with **role-based access** (Admin/User) to manage bank accounts efficiently.  
The system allows users to check balances, perform fund transfers, and track transaction history with proper validations.

---

## Key Features
- **Role-Based Access:** Admin and User functionalities  
- **Account Management:** Create and manage bank accounts  
- **Transactions:** Deposit, withdraw, and transfer funds  
- **Transaction History:** Maintain and track all operations  
- **Validation:** Ensures accurate and reliable transactions  

---

## Technologies
- Java  
- Object-Oriented Programming (OOP)  
- Java Collections Framework  
- Java Streams  

---

## How to Run
1. Clone or download this repository  
2. Open the project in your IDE (Eclipse, IntelliJ, or VS Code)  
3. Navigate to the `src` folder  
4. Run the `Main.java` file  
5. Follow the console prompts to interact with the system  

---

## Notes
- This is a **console-based application** (no GUI)  
- Designed to showcase **modular, maintainable, and efficient Java code**  
- Ideal for understanding **OOP concepts, collections, and basic banking operations**
